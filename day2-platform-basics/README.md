1. What is Salesforce Platform?
Salesforce Platform is a cloud-based platform that helps businesses manage customer relationships, automate processes, store data, and build applications. It provides tools for developers and admins to create apps without managing servers or infrastructure.
App, Object, Tab

2. What is an App,object,tab in Salesforce?
An App in Salesforce is a collection of related tabs, objects, and features designed for a specific business purpose. It helps users work in one organized workspace.
Example: Sales App, Service App, College Management App.
An Object in Salesforce is like a database table that stores information. Each object contains records and fields.
Example:
Account Object → stores company details
Contact Object → stores customer details
A Tab is the user interface that allows users to access objects, records, dashboards, or apps easily.
Example:
Accounts Tab
Contacts Tab
Reports Tab
