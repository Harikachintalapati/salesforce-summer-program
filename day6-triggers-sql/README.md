1. What is SOQL?
SOQL (Salesforce Object Query Language) is used to retrieve data from Salesforce objects. It is similar to SQL but designed specifically for Salesforce databases.
2. What is an Apex Trigger?
An Apex Trigger is a piece of code that automatically executes before or after changes happen to Salesforce records such as insert, update, delete, or undelete.
3. Difference
Flow 
No-code or low-code automation
Easier for admins
Good for simple automation
Trigger
Code based automation using apex
used by developers
best for complex logic
Before Trigger
Runs before saving record
Used to update field values
